服务地址
以下接口均基于该地址，所有读写类接口需要在请求头 X-API-Key 或 URL 参数 ?api_key= 传入 API Key。
对外接口（需 API Key）
GET /plugin-repo/api/plugins
列出仓库内所有插件及其所有版本信息。
请求示例：
参数：
  • api_key <Header: X-API-Key> 必填，可在「API Keys」页面创建。
GET /plugin-repo/api/download/{pluginName}/{version}
下载指定插件版本的压缩包（zip）。
请求示例：
路径参数：
  • pluginName <String> 必填，插件名称。
  • version <String> 必填，插件版本号。
管理接口（需登录管理员）
POST /plugin-repo/web/import
导入插件包，multipart/form-data 方式上传 zip 文件。
参数：
  • file <File> 必填，插件 zip 包。
DELETE /plugin-repo/web/plugin/{pluginName}/{version}
删除指定插件的某个版本。
路径参数：
  • pluginName <String> 必填
  • version <String> 必填
